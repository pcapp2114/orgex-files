INTRODUCTION
------------

Web Modify adds extra features

  1) Adds node id and uuid on node edit form
  2) On node edit form, hide promotion options setting tab
  3) Add the dashboard menu item and page to the admin toolbar
  4) Redirect the user login

REQUIREMENTS
------------

No special requirements.


INSTALLATION
------------

 * Install as you would normally install a contributed Drupal module.
   See: https://www.drupal.org/node/895232 for further information.


PERMISSIONS
-------------

 * Under permissions settings: /admin/people/permissions
   You must allow access to the dashboard for every role

 * There is no configuration needed
